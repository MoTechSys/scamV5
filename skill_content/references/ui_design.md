# تحليل رسومات الواجهات لمشروع S-ACM

## 1. ملخص الواجهات المطلوبة

### الواجهات المشتركة (للجميع):
1.  الواجهة الترحيبية
2.  تسجيل الدخول
3.  تفعيل الحساب (4 مراحل)
4.  استعادة كلمة المرور

### واجهات الطالب:
1.  الواجهة الرئيسية (Dashboard)
2.  مقرراتي (عرض الملفات + ميزات الذكاء الاصطناعي)
3.  التقارير
4.  الملف الشخصي
5.  الإشعارات
6.  الإعدادات

### واجهات المدرس:
1.  الواجهة الرئيسية (Dashboard)
2.  رفع ملفات المقررات
3.  إرسال الإشعارات
4.  التقارير
5.  الملف الشخصي
6.  الإعدادات

### واجهات المسؤول (Admin):
1.  لوحة التحكم الرئيسية (Dashboard)
2.  إدارة المستخدمين
3.  إدارة الأدوار والصلاحيات
4.  إدارة المقررات
5.  التقارير الإدارية
6.  الإعدادات
7.  الملف الشخصي

---

## 2. ملاحظات التصميم العامة

1.  **التصميم موحد**: جميع الواجهات تستخدم نفس التخطيط (Navbar + Sidebar + Content Area).
2.  **القائمة الجانبية ديناميكية**: تتغير عناصر القائمة الجانبية بناءً على دور المستخدم وصلاحياته.
3.  **لوحة الألوان**: استخدام درجات اللونين الأزرق والأخضر بشكل أساسي، مع تصميم يدعم اللغة العربية (RTL).
4.  **البطاقات الإحصائية**: عنصر متكرر في جميع لوحات التحكم لعرض البيانات الهامة بسرعة.
5.  **الجداول**: تصميم موحد للجداول مع فلاتر بحث، ترقيم، وأزرار إجراءات (تعديل، حذف).
6.  **الأيقونات**: استخدام أيقونات واضحة من مكتبة Bootstrap Icons لتحسين تجربة المستخدم.

---

## 3. تحليل تفصيلي لواجهات محددة

### لوحة التحكم الرئيسية (Dashboard)

- **المحتوى**: تختلف البطاقات الإحصائية والرسوم البيانية المعروضة بناءً على دور المستخدم.
  - **الطالب**: يعرض عدد المقررات الحالية، المقررات المؤرشفة، والإشعارات الجديدة.
  - **المدرس**: يعرض عدد المقررات المعينة، الملفات المرفوعة، وإجمالي التحميلات.
  - **المسؤول**: يعرض إجمالي المستخدمين، المقررات، الملفات، والإشعارات، بالإضافة إلى رسم بياني لنمو المستخدمين.

### صفحة إدارة المستخدمين (للمسؤول)

- **الميزات**: 
  - زر "إضافة مستخدم جديد".
  - فلاتر بحث متقدمة (بالاسم، الدور، التخصص، المستوى، الحالة).
  - جدول يعرض قائمة المستخدمين مع معلوماتهم الأساسية وأزرار الإجراءات.

### صفحة إدارة الأدوار والصلاحيات (للمسؤول)

- **الميزات**:
  - زر "إضافة دور جديد".
  - جدول يعرض الأدوار الحالية والصلاحيات الممنوحة لكل دور على شكل Checkboxes، مما يسمح بتعديلها بسهولة.

### صفحة تفاصيل المقرر

- **المحتوى الديناميكي**:
  - **للطالب**: يرى قائمة بالملفات (محاضرات، واجبات، ملخصات) مع أزرار لاستخدام ميزات الذكاء الاصطناعي (تلخيص، توليد أسئلة).
  - **للمدرس**: يرى نفس قائمة الملفات مع أزرار إضافية لتعديل الملفات، حذفها، أو تغيير حالتها (مرئي/مخفي).
